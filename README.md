## ğŸ¯ Learning Objectives

This project covers essential HTML concepts:

- **Basic HTML Structure**: DOCTYPE, html, head, body elements
- **Text Formatting**: Bold, italic, underline, emphasis, highlighting
- **Styling**: Inline CSS, colors, fonts, backgrounds
- **Tables**: Complex table structures with rowspan/colspan
- **Images**: Image display, sizing, and alt attributes
- **Hyperlinks**: Text and image links with target attributes
- **Image Maps**: Interactive clickable regions on images
- **Special Characters**: HTML entities and symbols

## ğŸ“„ File Descriptions

### First.html
- **Purpose**: Introduction to HTML basics
- **Features**:
  - Animated marquee with styled heading
  - Inline CSS styling examples
  - Text formatting demonstrations
  - Comments with CSS property explanations

### TextFormatting.html
- **Purpose**: Comprehensive text formatting showcase
- **Features**:
  - All major text formatting tags (b, strong, em, i, u, mark, del, ins)
  - Subscript and superscript examples
  - Blockquote usage
  - Preformatted text with marquee animation
  - Various font styling techniques

### ques2.html
- **Purpose**: Advanced HTML features demonstration
- **Features**:
  - ASCII art star patterns using `<pre>` tag
  - Interactive image map of India
  - Clickable regions linking to Wikipedia
  - Complex coordinate mapping

### Ques3.html
- **Purpose**: Advanced table creation
- **Features**:
  - Multi-level table headers
  - Rowspan and colspan attributes
  - Course and subject data organization
  - Internal/external marking system
  - HTML entities (checkmarks)

### Ques4.html
- **Purpose**: Image hyperlinks and layout
- **Features**:
  - 3x4 grid of brand logos
  - Image hyperlinks to external sites
  - Uniform image sizing (200x200px)
  - Table-based layout structure
